My work as a Software developer.

Elory Tag Manager

Elory - short for Elephant Memory - is a file tagging program to help you keep track of files on your system by assigning them custom defined tags. Built with Python, SQLite, and Kivy as its GUI interface.

MrBeat Demo Program

A simple music sampling program. Load any kind of sound sample into the kit, activate samples at different "steps", set the BPM and hear your creation come to life! Created as part of a comprehensive Python tutorial. Implemented with Python and Kivy framework, utilizing SDL2 as the audio driver.

MerchantCo

An application for managing a small business' inventory and sales. Built with Django and PostGres, confined in a Docker container for deployment anywhere.
